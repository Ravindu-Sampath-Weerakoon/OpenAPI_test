Vehicle Body Control API Documentation

Comprehensive field-by-field reference for the Vehicle Body Control system.

Wi-Fi Station Mode JSON API Port 80

Full JSON Example

{
  "module": "Door, Centre Lock, Power Mirror and Rear Boot Door Control",
  "status": "success",
  "message": "Vehicle body control data retrieved successfully",
  "driveStatus": "SAFE_TO_DRIVE",
  "riskMessage": "No body control risk detected.",
  "data": {
    "front_left_door_status": "closed",
    "front_right_door_status": "closed",
    "rear_left_door_status": "closed",
    "rear_right_door_status": "closed",
    "centre_lock_status": "locked",
    "power_mirror_left_position": 12,
    "power_mirror_right_position": -5,
    "rear_boot_door_status": "closed",
    "error_status": "OK"
  },
  "error": {
    "code": 0,
    "message": "None"
  }
}

Variable Dictionary (Root Level)

  • moduleString
    The identification name for this specific submodule.
  • statusString
    The high-level execution status of the request (e.g., "success" or "error").
  • messageString
    A human-readable message describing the overall result.
  • driveStatusString
    Safety indicator. SAFE_TO_DRIVE means vehicle is safe; RISK_DETECTED means a safety issue was found.
  • riskMessageString
    Details regarding why a risk was detected (e.g., specific door open).

Variable Dictionary (Data Object)

  • front_left_door_statusString
    Current state of the front-left door. Values: "closed", "open".
  • front_right_door_statusString
    Current state of the front-right door. Values: "closed", "open".
  • rear_left_door_statusString
    Current state of the rear-left door. Values: "closed", "open".
  • rear_right_door_statusString
    Current state of the rear-right door. Values: "closed", "open".
  • centre_lock_statusString
    State of the central locking mechanism. Values: "locked", "unlocked".
  • power_mirror_left_positionInteger
    Left mirror offset from neutral. Positive = Up/Right, Negative = Down/Left, 0 = Center.
  • power_mirror_right_positionInteger
    Right mirror offset from neutral. Positive = Up/Right, Negative = Down/Left, 0 = Center.
  • rear_boot_door_statusString
    Current state of the trunk/boot. Values: "closed", "open".
  • error_statusString
    Internal sensor health status from the ESP32 (e.g., "OK").

Variable Dictionary (Error Object)

  • codeInteger
    Error code. 0 indicates no error.
  • messageString
    Specific error description. "None" if healthy.
