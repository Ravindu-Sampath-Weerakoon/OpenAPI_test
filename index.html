Vehicle Body Control API Documentation

This documentation is for the Door, Centre Lock, Power Mirror and Rear Boot Door Control module, developed using an ESP32 configured in Station (STA) Mode. The ESP32 connects to a Wi-Fi network and hosts a web server on port 80.

Wi-Fi Station Mode Dynamic IP Addressing Port: 80 Response Type: JSON

Integration Steps

  1. Power on the ESP32 vehicle submodule.
  2. Ensure it connects to the Wi-Fi network (SSID: Dialog 4G 048).
  3. Open the ESP32 Serial Monitor to find the Assigned IP Address.
  4. Send an HTTP GET request to http://<ESP32_IP>/.
  5. Read the returned JSON response and check driveStatus.

Safety Rule

Type Meaning Controller Action
SAFE_TO_DRIVE Module works normally and vehicle is safe. Continue operation.
RISK_DETECTED A safety risk is detected (e.g., door open). Stop or alert the user.
